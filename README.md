# job_application_tracker
Claude agent for application tracking. Uses Obsidian WebClipper for automated information input

# Usage
Copy CLAUDE.md file into working directory. Claude will create all subdirectories upon first prompt. 

# Obsidian WebClipper
Download extention from Chrome and under Settings -> Edit Template -> Location -> Note location, set the default clipping directory to /raw/clippings

Select extension in the webpage with your job's JD. It will autofill the details and prompt you to add it to obsidian. Select "Add to Obsidian" and the clipping entry will appear in your directory folder. 
Prompt Claude to ingest new entries and it will update both the tracker and Obsidian Vault. You can view jobs applied by Sector and Company in Graph View.
